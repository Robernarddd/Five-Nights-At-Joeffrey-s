# PROJET : FNAF ENTRE POTES

## OBJECTIF

Créer un jeu d'horreur humoristique inspiré de Five Nights at Freddy's 1.

Le jeu doit être jouable directement dans un navigateur web.

Le rendu doit être propre et sérieux malgré l'aspect troll des personnages.

Le gameplay doit être très proche de FNAF 1.

Les animatroniques originaux seront remplacés par des amis du créateur sous forme d'images PNG personnalisées.

---

# TECHNOLOGIES

- HTML5
- CSS3
- JavaScript Vanilla

Pas de framework obligatoire.

Le projet doit rester simple à comprendre pour un développeur débutant.

# GAMEPLAY

Le joueur est assis dans un bureau.

Il doit survivre de 00:00 à 06:00.

Une nuit dure environ 6 à 8 minutes réelles.

Le joueur dispose de :

- Porte gauche
- Porte droite
- Lumière gauche
- Lumière droite
- Caméras de surveillance

Les actions consomment de l'énergie.

Lorsque l'énergie atteint zéro :

- Coupure de courant
- Séquence de défaite
- Game Over
